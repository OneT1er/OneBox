# 更新日志 / Changelog

## v1.1.0

### 新增
- **剪贴板支持图片**: 复制图片(截图/图片)自动记录,历史列表显示缩略图,点击复制回图片
- **快捷启动栏拖拽**: 可直接把 exe 或快捷方式(.lnk)拖入格子添加,自动解析快捷方式目标路径
- **剪贴板历史**: 文本+图片混合记录,持久化,点击即复制
- **统一窗口样式**: 设置对话框/剪贴板面板改用 OneBox 深色风格
- **自动下载安装更新**: 检查到新版本可直接"立即更新",自动下载替换重启

### 修复
- 电源计划增删后音频设备名称消失(powercfg 加超时 + 数据获取失败时保留旧数据)
- LoadData 卡死导致后续刷新全被跳过(_loading 加 10 秒超时保护)
- 托盘菜单勾选项不显示(ShowImageMargin 恢复)
- 分辨率切换(4K↔1080P)后窗口找不到(Screen.Bounds + 2 秒轮询兜底)
- 翻译窗 ComboBox 白字灰底看不清 + 点不了(自定义深色控件模板)
- UpdateChecker TLS 1.2 / C#5 语法修正

### 变更
- **窗口按屏幕比例缩放**: 1080P(1.0x) / 4K(1.5x),分辨率切换自动适配
- **资源嵌入 exe**: 字体、图标全部嵌入,输出目录只剩 OneBox.exe 单文件
- **去掉 Per-Monitor DPI**: 改为系统级 DPI 感知,简化分辨率适配
- **去掉 OneBox.exe.config**: DPI 开关移除后已无用

### 移除
- 外部字体/图标文件(全部嵌入 exe)
- OneBox.exe.config
